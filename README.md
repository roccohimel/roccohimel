# 💫 About Me:
’m a programmer and builder who’s really into low-level systems, operating systems, and how computers actually work under the hood. I spend a lot of my time writing C, experimenting with kernels, and building my own projects like custom terminal environments and OS concepts. I like keeping things minimal, fast, and close to the hardware.
